## Requirements

NodeJS 10^

## Install all depencies

```bash
npm i
```

## Development

```bash
npm run dev
```

## Production static (copy all files from ./dist to server)

```bash
npm run build
```

## Production with server (Start application on node server)

```bash
npm run build:full
npm run start
```
